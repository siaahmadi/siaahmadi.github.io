a post with chat conversations

Created on January 15, 2024

2024   ·   formatting   ·   sample-posts

The chat_message include lets you embed clean, iMessage-style conversation threads directly in your posts — great for showcasing LLM interactions, dialogue examples, interview excerpts, or any back-and-forth exchange.

Basic usage

Wrap one or more {% include chat_message.html %} calls inside a <div class="chat-container">:

<div class="chat-container">
  {% include chat_message.html role="user" name="You" content="What is the capital of France?" %}
  {% include chat_message.html role="assistant" name="Claude" content="The capital of France is **Paris**." %}
</div>

Which renders as:

You

What is the capital of France?

Claude

The capital of France is Paris.

Multi-turn conversation

You

Can you explain what a transformer neural network is in plain English?

Claude

Sure! Think of a transformer as a very sophisticated attention machine. Instead of reading a sentence word-by-word from left to right (like older models did), it looks at all the words at once and figures out which ones are most relevant to each other.

You

So it’s like reading the whole page before deciding what’s important?

Claude

Exactly — that’s a great analogy. The “attention” mechanism lets every word vote on how much it should pay attention to every other word in the sentence. That’s why transformers are so good at capturing long-range dependencies in text.

With timestamps

Pass a time parameter to show a timestamp beneath each bubble:

Alice

Are you free for lunch on Friday?

10:14 AM
Bob

Yes! Where were you thinking?

10:16 AM
Alice

That new ramen place on 5th?

10:17 AM
Bob

Perfect, see you at noon 🍜

10:18 AM

Without author names

Omit name for a cleaner iMessage-style look:

Remind me: what did we decide about the API design?

We agreed to use REST with JWT authentication for v1, and revisit GraphQL once the team has grown.

Right, thanks.

Using the divider

Use <div class="chat-divider">Label</div> to separate sections of a conversation:

You

Can you write a haiku about the ocean?

Claude

Waves crash endlessly,\nSalt and foam kiss the cold shore,\nThe deep holds its breath.

Later that day
You

Now one about mountains?

Claude

Stone older than words,\nClouds break against the summit,\nSilence fills the gaps.

Markdown inside bubbles

Message content is passed through the Markdown processor, so you can use bold, italic, inline code, and even links:

You

What’s the Python syntax for a list comprehension?

Claude

The basic syntax is [expression for item in iterable]. For example, [x**2 for x in range(10)] gives you the squares of 0–9. You can also add a filter: [x for x in range(10) if x % 2 == 0].

Include parameters reference

Parameter Required Values Description
role ✓ "user" / "assistant" Controls which side the bubble appears on
content ✓ any string (Markdown OK) The message text
name – any string Author label above the bubble
time – any string Timestamp below the bubble

The role parameter is flexible: "user" and "right" both produce a right-aligned bubble; everything else (including "assistant", "bot", "left", or any custom string) produces a left-aligned bubble.
